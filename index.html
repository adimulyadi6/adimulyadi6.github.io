About Experience Projects

Saya merupakan mahasiswa semester 8 jurusan Teknik Informatika dengan minat utama pada bidang pengembangan web. Ketertarikan ini mendorong saya untuk memahami berbagai konsep dasar hingga lanjutan dalam pengembangan aplikasi berbasis web, baik dari sisi frontend maupun dasar-dasar backend. Saya juga terbiasa mempelajari teknologi terbaru untuk mengikuti perkembangan industri yang dinamis.

Selama masa perkuliahan, saya telah memiliki pengalaman dalam membangun beberapa aplikasi berbasis web, mulai dari perancangan tampilan antarmuka hingga implementasi fitur fungsional. Dalam proses tersebut, saya menggunakan teknologi seperti HTML, CSS, dan JavaScript, serta memahami pentingnya membuat aplikasi yang responsif, efisien, dan mudah digunakan oleh pengguna.

Saya memiliki semangat belajar yang tinggi dan komitmen untuk terus mengembangkan keterampilan melalui pengalaman nyata di dunia kerja, khususnya melalui program magang. Dengan pengalaman tersebut, saya berupaya meningkatkan kemampuan teknis, memperkuat kerja sama tim, serta memahami alur kerja profesional dalam pengembangan perangkat lunak secara langsung.
